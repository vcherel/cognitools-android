# BestApp

An Android app built in Kotlin for **fun**, designed to group all the little tools and games I like to have on my phone.

### Features
- **Random Number & Word Generator**

Quickly generate random numbers or words whenever you need a little randomness in your day.

- **Volume Booster**

Increase or decrease your phone's sound volume beyond the default limits.

- **Flashcards for Learning**

Learn word pairs efficiently. The app intelligently shows cards we know less often, so you focus on what you need to remember.

- **Undercover Game**

Play a social deduction game where each player receives a secret word. Most players get the same word, but one or multiple ***Undercover*** have a different word and/or one or multiple ***Mr. White*** donâ€™t get any word at all. Players take turns giving clues about their word, trying not to reveal it. The goal is to identify the undercovers while undercovers try to blend in.

The pairs of words used in the game were generated automatically using **NLP** and **FastText embeddings**. Using this method, the game includes more than 280,000 pairs of related words.

- **Wikipedia Reader**

Explore random Wikipedia pages
